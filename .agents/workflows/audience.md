# /audience — Audience Builder

## When to Use
Use `/audience` when creating new targeting, refining existing audiences, or uploading CRM data.

## Steps

1. **Determine Type** — New from scratch? Refine existing? Lookalike? CRM upload?

2. **Load Context** — Read `skills/audience-targeting/SKILL.md`, `brain/CLIENT.md`, `memory/CAMPAIGN_TRACKER.md`.

3. **Build Audience:**

   **Google Ads:**
   - Keyword list with match types (broad, phrase, exact)
   - Location targeting (licensed states only)
   - Demographics, in-market audiences, negative keywords

   **Meta Ads (CREDIT restrictions):**
   - Location only (15-mile min radius, NO age/gender/zip)
   - Interests/behaviors
   - Custom audience: pixel visitors, CRM upload (SHA-256 hashed)
   - Lookalike: 1-5% from converting customers

4. **Estimate Reach** — Query platform for audience size.

5. **Apply** — Link to campaign, log to tracker.

## Example Usage
**User:** `/audience`
**Agent:** "What type? New from scratch, refine existing, lookalike, or CRM upload?"
